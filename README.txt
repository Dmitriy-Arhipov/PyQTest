Юридические аспекты
Я сторонник opensource – использование моего приложения бесплатно для всех желающих, а исходный код открыт для чтения и редактирования.
При этом разрешаются любые его изменения, но я снимаю с себя ответственность за возможный вред, причиненный использованием измененной кем-либо версии программы. 
Платное распространение и какое-либо коммерческое использование даже измененной версии не допускается.
Использование приложения означает согласие с вышеизложенной политикой.
